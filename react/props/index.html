Aula sobre props no React

1. O que são props?

props é a abreviação de "propriedades" e são usados para passar dados de um componente pai para um componente filho no React.

Elas permitem que componentes sejam reutilizáveis e dinâmicos.

function Saudacao(props) {
  return <h1>Olá, {props.nome}</h1>;
}

2. Como as props funcionam?

As props são passadas para um componente como atributos em JSX e acessadas dentro do componente através de props. 

function App() {
  return <Saudacao nome="Maria" />;
}

3. Props são somente leitura

Uma regra importante sobre props é que elas são imutáveis, ou seja, o componente filho não deve modificá-las.

Elas servem apenas para leitura. Para alterar dados, use o state em vez disso.

4. Passando múltiplas props

Você pode passar quantas props quiser para um componente. Elas serão acessadas como propriedades no objeto props. 

function Saudacao(props) {
  return <p>Olá, {props.nome}. Você tem {props.idade} anos.</p>;
}

function App() {
  return <Saudacao nome="João" idade={30} />;
}

5. Props padrão (default props)

Você pode definir valores padrão para props caso nenhuma seja passada. 

function Saudacao(props) {
  return <p>Olá, {props.nome}!</p>;
}

Saudacao.defaultProps = {
  nome: 'Visitante'
};

function App() {
  return <Saudacao />; // Saudação padrão: "Olá, Visitante!"
}

6. Passando funções como props

É possível passar funções como props para componentes. Isso permite, por exemplo, que o componente filho execute uma ação no componente pai. 

function Botao(props) {
  return <button onClick={props.aoClicar}>Clique aqui</button>;
}

function App() {
  const mostrarAlerta = () => {
    alert('Botão clicado!');
  };

  return <Botao aoClicar={mostrarAlerta} />;
}

7. Desestruturando props

Em vez de acessar as props como props.nome, você pode desestruturá-las diretamente nos parâmetros da função. 

function Saudacao({ nome, idade }) {
  return <p>Olá, {nome}. Você tem {idade} anos.</p>;
}

function App() {
  return <Saudacao nome="Ana" idade={25} />;
}

8. Exemplos práticos de componentes com props

Exemplo de um componente que renderiza uma lista de itens recebidos como props: 

function ListaDeItens({ itens }) {
  return (
    <ul>
      {itens.map((item, index) => (
        <li key={index}>{item}</li>
      ))}
    </ul>
  );
}

function App() {
  const meusItens = ['Item 1', 'Item 2', 'Item 3'];
  return <ListaDeItens itens={meusItens} />;
}

9. Boas práticas no uso de props

  • Use props para tornar componentes reutilizáveis.
  • Desestruture props para deixar o código mais limpo.
  • Defina valores padrão para props sempre que possível.
