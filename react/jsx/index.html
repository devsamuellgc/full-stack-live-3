Aula sobre JSX no React

1. O que é JSX?

JSX (JavaScript XML) é uma extensão de sintaxe para JavaScript usada no React. Ele permite escrever código que se parece com HTML dentro de arquivos JavaScript. 

const element = <h1>Hello, world!</h1>;

JSX é compilado para React.createElement(), que retorna um objeto JavaScript representando um elemento DOM.

2. Vantagens do JSX

  • Intuitivo e legível: Facilita o desenvolvimento, especialmente para quem já conhece HTML.
  • Segurança: O React escapa automaticamente dados no JSX, prevenindo ataques XSS.
  • Expressividade: Você pode usar expressões JavaScript dentro do JSX. 
const name = 'Maria';
const element = <h1>Hello, {name}!</h1>;

3. Como o JSX é transformado?

JSX é transformado em JavaScript através de ferramentas como Babel.

// JSX
const element = <h1>Hello, world!</h1>;

// JavaScript compilado
const element = React.createElement('h1', null, 'Hello, world!');

4. Diferenças entre HTML e JSX

  • CamelCase: Atributos no JSX são escritos em camelCase (className ao invés de class).
  • Fechamento de tags: Todas as tags precisam ser fechadas, mesmo aquelas sem conteúdo.
  • Fragmentos: Use <></> para agrupar elementos sem criar um nó adicional no DOM. 
const element = (
  <>
    <h1>Hello!</h1>
    <p>Welcome to JSX.</p>
  </>
);

5. Incorporando Expressões JavaScript no JSX

Você pode usar expressões JavaScript dentro do JSX, como variáveis, operadores ternários, etc. 

const user = { firstName: 'João', lastName: 'Silva' };
const greeting = <h1>Hello, {user.firstName} {user.lastName}!</h1>;

6. Estilos e Classes no JSX

Estilos inline: Passados como objetos JavaScript com propriedades em camelCase.

Classes: Use className em vez de class. 

const element = (
  <div style={{ color: 'blue', backgroundColor: 'lightgray' }}>
    This is a styled element!
  </div>
);

7. Funções e Componentes em JSX

Componentes em React podem ser criados como funções que retornam JSX.

function Welcome(props) {
  return <h1>Hello, {props.name}</h1>;
}

const element = <Welcome name="João" />;

8. Boas Práticas no Uso de JSX

  • Divida componentes grandes: Se o componente fica muito longo, divida-o em componentes menores.
  • Modere o uso de expressões complexas: Prefira guardar valores intermediários em variáveis. 
const total = 5 + 10; // Melhor calcular fora do JSX
return <p>The total is {total}</p>;

9. Hands-on com JSX (Atividade prática)

Peça aos alunos para criar um componente que renderiza uma lista de itens dinamicamente. 

function ShoppingList() {
  const items = ['Apple', 'Banana', 'Cherry'];
  return (
    <ul>
      {items.map((item, index) => (
        <li key={index}>{item}</li>
      ))}
    </ul>
  );
}
